## About Me

I'm Da, a Computer Engineering student based in Genova, Italy, building an engineering-driven career that blends AI, cloud, cybersecurity, and music technology into real-world products and creative experiences.

I’m currently deep into Artificial Intelligence, Machine Learning, neural networks, generative AI, and AI agents, with a strong focus on creative applications for music, video, and digital content creation.

On the engineering side, I enjoy full-stack development, cloud-native systems, and automation workflows; on the creative side, I'm growing my music career as **NokMusic**, building tools that power production, mixing, and artist-centric digital experiences.

**Open To**: AI/ML engineering internships, cloud & cybersecurity projects, creative AI collaborations, and full-stack product engineering roles.

## Featured Projects

<details>
  <summary><b>NokMusic AI Studio — AI-Powered Music Workflow Engine</b></summary>

  A modular AI studio that connects DAWs, cloud ML APIs, and local LLM agents to automate parts of the music production workflow (idea generation, stems management, metadata, and release prep).

  | Metric | Details |
  |--------|---------|
  | Stack | Python, FastAPI, GCP (Vision / NLP / Translation), local LLMs (Gemma / Ollama), Docker |
  | Scale | Designed for independent artists and small labels; multi-project, multi-artist pipeline |
  | Performance | Async processing for batch audio tasks; background job orchestration for long-running workflows |
  | Security | API key management, IAM roles, least-privilege cloud access, secrets handling |
  | Impact | Speeds up repetitive production tasks; helps artists stay in the creative zone |
  | Repository | Coming soon on GitHub as a public project with modular plugin-like architecture |

</details>

<details>
  <summary><b>VisionFlow — Real-Time Motion Capture & Gesture Engine</b></summary>

  A computer vision system for markerless motion capture, real-time pose tracking, and gesture recognition, built for interactive experiences, performance analysis, and creative media.

  | Metric | Details |
  |--------|---------|
  | Stack | Python, OpenCV, MediaPipe, WebSockets, React frontend |
  | Scale | Real-time processing for live streams and recorded sessions |
  | Performance | Optimized pipelines with frame skipping, batching, GPU-aware execution where available |
  | Security | Local-first processing; optional encrypted streaming for remote sessions |
  | Impact | Enables interactive installations, performance tracking, and motion-based creative tools |
  | Repository | Experimental repo with modular vision pipelines and pluggable outputs |

</details>

<details>
  <summary><b>CloudSec Sentinel — Cloud Security & Threat Monitoring Lab</b></summary>

  A hands-on lab project for learning and applying cloud security, IAM, logging, and basic threat detection in GCP and AWS.

  | Metric | Details |
  |--------|---------|
  | Stack | GCP, AWS, Python, Terraform (experimental), Logging / Monitoring APIs |
  | Scale | Small to medium cloud workloads; focused on educational and portfolio scenarios |
  | Performance | Event-driven detection flows; efficient log queries and dashboards |
  | Security | Hardening IAM, role-based access control, principle of least privilege |
  | Impact | Demonstrates practical cloud security skills and helps build deployable security patterns |
  | Repository | Cloud security lab documentation and sample configurations |

</details>

<details>
  <summary><b>AgentLab — Local-First AI Agent Orchestration</b></summary>

  A local-first AI agent playground for orchestrating multiple LLMs and tools, focusing on reproducible workflows, specs-driven development, and offline-capable systems.

  | Metric | Details |
  |--------|---------|
  | Stack | Python, Docker, local LLMs (Ollama), REST APIs, CLI tooling |
  | Scale | Multi-agent experiments; extensible architecture for new tools and models |
  | Performance | Lightweight orchestration with caching and streaming outputs |
  | Security | Local-first execution, controlled API exposure, token management |
  | Impact | Showcases system design skills and practical AI agent engineering |
  | Repository | Public repo planned as part of portfolio-focused open source contributions |

</details>
