Combines academic foundations in computer engineering with hands-on experience designing AI agents, automating engineering workflows, and applying secure-by-design principles across the full stack, from cloud infrastructure to application logic.

Committed to writing maintainable, well-documented, and testable code, with an engineering mindset oriented toward reliability, scalability, and real-world product impact.

## Featured Project

<details open>
  <summary><b>🔐 secure-review-agent — AI-Powered Secure Code Review Agent</b></summary>

  <br>

  An AI agent designed to automate secure code review, identifying vulnerabilities, insecure patterns, and security risks directly within the development workflow.

  | Metric | Details |
  |--------|---------|
  | Stack | Python, LLM-based analysis, local model integration (Ollama / Llama) |
  | Scale | Designed to run per-repository and integrate into CI/CD security review workflows |
  | Performance | Automated static-style security review with LLM-driven reasoning over code changes |
  | Security | Core purpose is proactive vulnerability and secure-code-pattern detection |
  | Impact | Reduces manual security review overhead and helps enforce secure coding practices |
  | Repository | [github.com/Dacfor/secure-review-agent](https://github.com/Dacfor/secure-review-agent) |
